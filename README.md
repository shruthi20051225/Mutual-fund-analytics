# Mutual Fund Analytics Capstone

## Overview

This project is an end-to-end mutual fund analytics solution built using Python, SQLite, Jupyter Notebook, and Power BI. It covers data ingestion, cleaning, exploratory analysis, performance analytics, advanced risk analytics, and dashboard visualization.

## Technologies Used

- Python
- Pandas
- NumPy
- Plotly
- Matplotlib
- SQLite
- Jupyter Notebook
- Power BI Desktop
- Git & GitHub

## Project Workflow

1. Data Ingestion
2. Data Cleaning & Validation
3. SQLite Database Creation
4. Exploratory Data Analysis (EDA)
5. Performance Analytics
6. Advanced Analytics
7. Power BI Dashboard
8. Final Reporting

## Setup

Clone the repository:

```bash
git clone https://github.com/shruthi20051225/Mutual-fund-analytics.git
```

Create virtual environment:

```bash
python -m venv venv
```

Activate environment:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run pipeline:

```bash
python run_pipeline.py
```
